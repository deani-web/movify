# Movify

Movify adalah aplikasi berbasis Flutter yang dirancang untuk menjelajahi dan menampilkan informasi film secara interaktif. Aplikasi ini terhubung dengan API untuk menyajikan data film secara *real-time* dan sudah dikonfigurasi agar dapat berjalan optimal di perangkat Android.

---

## 📱 Download & Install (APK)

Aplikasi ini sudah siap pakai dan bisa langsung diinstal di perangkat Android kamu tanpa perlu melakukan *build* ulang dari *source code*.

1. Masuk ke folder bawaan projek ini di GitHub: **`build/app/outputs/flutter-apk/`**.
2. Klik pada file **`app-release.apk`**.
3. Klik tombol **Download** untuk mengunduh file APK ke komputer atau perangkat Android kamu.
4. Jika diinstal langsung di Android, pastikan kamu telah mengizinkan instalasi dari sumber tidak dikenal (*Unknown Sources*) di pengaturan keamanan HP kamu.

---

## 🚀 Fitur Utama

* **Eksplorasi Film:** Menampilkan daftar film terpopuler, terbaru, dan sedang tayang.
* **Data Real-Time:** Terintegrasi langsung dengan API eksternal untuk pembaruan data informasi film.
* **Antarmuka Interaktif:** Desain UI/UX yang modern, bersih, dan responsif khusus untuk perangkat *mobile*.
